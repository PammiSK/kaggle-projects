# Kaggle Projects

A collection of Kaggle projects completed to demonstrate skills in machine learning, deep learning, natural language processing, forecasting, and predictive analytics.

## Projects

| Project | Domain |
|----------|----------|
| Ultimate Customer Churn Prediction Challenge | Classification |
| LLM Classification Finetuning | NLP / LLM |
| Store Sales - Time Series Forecasting | Forecasting |
| House Prices - Advanced Regression Techniques | Regression |
| Digit Recogniser | Computer Vision |
| Titanic - Machine Learning from Disaster | Classification |
| Natural Language Processing with Disaster Tweets | NLP |
